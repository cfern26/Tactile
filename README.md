# Proyecto TactileJr / Scratch Tactile

Este proyecto es una iniciativa de **programación creativa inclusiva** que utiliza bloques físicos tangibles para enseñar pensamiento computacional a niños, especialmente a aquellos con discapacidad visual o motriz.

## Colaboradores Clave
- **MIT Media Lab (Lifelong Kindergarten):** Creadores originales de Scratch.
- **Fundación Scratch:** Apoyo y desarrollo global.
- **Sistema THEAD (Barcelona):** Organización sin ánimo de lucro que lidera el diseño físico y la implementación en Cataluña.
- **Ayuntamiento de Barcelona:** Institución colaboradora en el despliegue local.

## Referencias Principales
- **Scratch Tactile:** [scratchtactile.org](https://scratchtactile.org)
- **Movimiento TactileJr:** [tactilejr.org](https://tactilejr.org)
- **Investigación MIT:** Tesis de Tina Quach (2019) sobre interfaces conversacionales para niños ciegos.
